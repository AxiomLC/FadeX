# Template for Orderly SDK v2 based on Create React App

## Getting started

1. Clone the code:

```
git clone https://github.com/OrderlyNetwork/orderly-js-sdk-cra-template.git
cd orderly-js-sdk-cra-template
```

2. Install the dependencies

```
npm install
```

3. Run

```
npm start
```
